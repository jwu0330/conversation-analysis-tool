# data/ — 真實研究資料放這裡

資料依分析型態分成兩個子資料夾，避免混淆、也讓兩套分析互不干擾：

| 子資料夾 | 用途 | 對應模組 |
| --- | --- | --- |
| `data/對話分析/` | 逐題 Bloom 標記的對話紀錄（序列分析用） | 🔀 序列分析（`src/sequence`） |
| `data/量化/` | 學生前測／後測測驗結果（準實驗量化分析用） | 📈 準實驗分析（`src/quant`，規劃中） |

**對話分析**：把 Bloom 標記檔放進 `data/對話分析/`，會自動出現在序列分析的「內建資料」選單。

**量化分析**：把前後測成績檔放進 `data/量化/`，供準實驗分析模組讀取
（詳見 `docs/準實驗分析方法.md`）。

## 隱私

本資料夾內容已被 `.gitignore` 排除，**不會**被推上公開 GitHub / Streamlit Cloud，
只保留在你的本機。這是為了保護學生對話資料。若日後要部署含真實資料，請另行評估去識別化。

## 建議欄位（精簡後）

| 欄位 | 說明 |
| --- | --- |
| 學生ID | 學生識別碼 |
| 組別 | 實驗組 / 對照組 |
| Bloom層級 | L0–L6 或 Level 1–6 |
| 題序 / 時間 | 用來排序（同一學生的提問先後） |

其餘敘述、旁白等與分析無關的欄位可移除；日後若需要再從原始檔補回。
