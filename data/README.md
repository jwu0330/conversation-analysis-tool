# data/ — 真實研究資料放這裡

把你整理好的 Bloom 標記檔（.xlsx 或 .csv）放進這個資料夾，
系統會自動在「📤 資料上傳」與首頁的「內建資料」選單中出現（測試資料二、三…），
不需改任何程式。

## 隱私

本資料夾內容已被 `.gitignore` 排除，**不會**被推上公開 GitHub / Streamlit Cloud，
只保留在你的本機。這是為了保護學生對話資料。若日後要部署含真實資料，請另行評估去識別化。

## 建議欄位（精簡後）

| 欄位 | 說明 |
| --- | --- |
| 學生ID | 學生識別碼 |
| 組別 | 實驗組 / 對照組 |
| Bloom層級 | L0–L6 或 Level 1–6 |
| 題序 / 時間 | 用來排序（同一學生的提問先後） |

其餘敘述、旁白等與分析無關的欄位可移除；日後若需要再從原始檔補回。
